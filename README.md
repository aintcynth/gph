## üîß Customization

### Update Personal Information
Edit the following in `index.html`:
- Name and title in hero section
- About section content
- Education details
- Contact information
- Social media links

### Modify Styles
Edit `style.css` to customize:
- Colors: Update color values in `:root` or specific sections
- Fonts: Change font imports from Google Fonts
- Spacing: Adjust padding and margins in media queries

### Add Media
Replace image and video files in the `assets/` folder with your own.
